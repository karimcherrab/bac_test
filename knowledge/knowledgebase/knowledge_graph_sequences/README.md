# Knowledge Graph - Sequences

هذا الملف يربط كل مكتبات فصل المتتاليات:

- Concepts
- Skills
- Methods
- Formulas
- Theorems
- Examples
- Hints
- Common Mistakes

## الملفات

- `knowledge_graph_sequences.json`: الرسم البياني الكامل.
- `nodes.csv`: جميع العقد.
- `edges.csv`: العلاقات بين العقد.
- `graph_rag_summary.jsonl`: ملخص جاهز لمحرك RAG.

## مثال استعمال

إذا سأل الطالب: "كيف أثبت أن المتتالية متزايدة؟"

المسار يكون:

CONCEPT-004 → SKILL-SEQ-001 → METHOD-SEQ-002 → FORMULA-SEQ-001 → THEOREM-SEQ-011 → EXAMPLE-SEQ-001 → HINTS → MISTAKES
