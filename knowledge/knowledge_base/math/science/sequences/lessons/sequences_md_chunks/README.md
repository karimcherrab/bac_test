# تقسيم درس المتتاليات - شعبة علوم تجريبية

هذا المجلد يحتوي على تقسيم احترافي لدرس المتتاليات العددية إلى ملفات Markdown جاهزة للاستعمال في نظام RAG.

## المحتويات

- ملفات `.md`: كل ملف يمثل Chunk مستقل.
- `manifest_sequences_chunks.csv`: فهرس كل الـ Chunks.
- `chunks_sequences.jsonl`: نسخة جاهزة للـ ingestion في ChromaDB أو pgvector.

## طريقة الاستعمال في RAG

1. اقرأ ملفات Markdown أو ملف JSONL.
2. أنشئ embedding لكل `text`.
3. خزّن النص + metadata في قاعدة vector.
4. عند سؤال الطالب، ابحث في `chapter=المتتاليات العددية` و `branch=علوم تجريبية`.

## ملاحظة

هذا تقسيم تعليمي دلالي وليس مجرد تقسيم حسب الصفحات.
