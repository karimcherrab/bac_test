# مكتبة Methods لفصل المتتاليات

هذه المكتبة مبنية على تمارين البكالوريا 2008–2015 التي قمنا بتقسيمها إلى Learning Objects.

## المحتوى

- عدد Methods: 15
- عدد Chunks الجاهزة لـ RAG: 30
- الشعبة: علوم تجريبية
- المادة: رياضيات
- الفصل: المتتاليات

## لكل Method

- `method.md`: الشرح الكامل للطريقة.
- `rag_summary.md`: ملخص سريع لمحرك RAG.
- `skills.json`: مفاهيم، مهارات، صعوبة، أسئلة مرتبطة.
- `references.json`: روابط إلى أسئلة البكالوريا التي تستعمل هذه الطريقة.

## كيف يستعملها النظام؟

إذا سأل الطالب:

"كيف أثبت أن المتتالية متزايدة؟"

يبحث النظام أولًا في:
- `type = method`
- `concept = monotonicity`

ثم يجلب:
- طريقة إثبات الرتابة.
- أسئلة البكالوريا المرتبطة.
- الأخطاء الشائعة.

بعدها يعطي إجابة تعليمية وليس مجرد حل.
