# استعمال مبرهنة الحصر لحساب النهاية

## متى أستعملها؟
عندما يعطي السؤال علاقة من الشكل:

\[
0 \le A_n \le B_n
\]

وكان:

\[
B_n \to 0
\]

فإن:

\[
A_n \to 0
\]

## المثال الشائع
إذا ثبت:

\[
0\le 6-u_n\le v_n
\]

و:

\[
v_n\to0
\]

فإن:

\[
6-u_n\to0
\]

ومنها:

\[
u_n\to6
\]

## الخوارزمية

1. أثبت المتراجحة الثنائية.
2. احسب نهاية الطرف السهل.
3. طبق مبرهنة الحصر.
4. استنتج نهاية المتتالية المطلوبة.

## أخطاء شائعة
- نسيان شرط الطرفين.
- استنتاج النهاية بدون إثبات أن الطرف الأكبر يؤول إلى 0.
- الخلط بين \(6-u_n\to0\) و \(u_n\to0\).
